EXACT FILES TO UPLOAD TO GITHUB (File by File)
===============================================

STEP 1: Upload these ROOT FILES first:
---------------------------------------
1. package.json
2. package-lock.json
3. README.md (optional)

STEP 2: Create PUBLIC folder structure and upload files:
--------------------------------------------------------
Create folder: public/

Then upload these files INTO the public/ folder:
- public/index.html
- public/favicon.ico
- public/manifest.json
- public/robots.txt
- public/logo192.png
- public/logo512.png
- public/test.html

Create subfolder: public/brand/
Upload into public/brand/:
- public/brand/logo-placeholder.txt
- public/brand/no-choice-logo.svg

Create subfolder: public/gallery/ (if it has files)
Upload any files from public/gallery/ if they exist

IMPORTANT NOTES:
- When creating folders in GitHub, click "Add file" → "Create new file"
- Type the folder name followed by a filename, like: public/index.html
- GitHub will automatically create the folder structure
- Upload files one at a time or use "Add file" → "Upload files" for multiple
