# No Choice Beauty Salon Website

A modern, elegant website for No Choice Beauty Salon featuring a polaroid-inspired design.

## Features

- Responsive design for mobile, tablet, and desktop
- Services section with category selector (Nails, Makeup, Eyelashes, Eyebrows)
- Booking integration with Vagaro for both locations
- Contact information for Huntington Beach and Los Angeles locations
- Reviews section
- Gallery showcase

## Tech Stack

- React 18
- Material-UI (MUI)
- React Router
- Custom fonts (Caveat, Crimson Text)

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

Runs the app in development mode at [http://localhost:3000](http://localhost:3000)

### Build for Production

```bash
npm run build
```

Builds the app for production to the `build` folder.

## Deployment

This site is configured for deployment on Netlify.

### Netlify Build Settings

- **Build command:** `npm run build`
- **Publish directory:** `build`
- **Base directory:** (leave blank if repo is frontend folder)
